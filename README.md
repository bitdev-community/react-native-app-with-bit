# react-native-app-with-bit

Install dependencies:

``` bash
npm install
```

Run the app locally:

``` bash
npm start
```

Run Bits' dev environment:

``` bash
bit start
```

Create new components:

``` bash
bit create react-native <component-name>
```

Tag components:

``` bash
bit tag -m "my message"
```

Export components to bit.cloud:

``` bash
bit export 
```
