import { Spinner } from './spinner';

The Spinner component provides a visual indication of loading state in your application. It can be customized with different sizes, colors, and an optional accompanying text.

### Component Usage

```ts live
function LoadingComponent() {
  return <Spinner size="large" color="blue" text="Loading data..." />;
}
```
